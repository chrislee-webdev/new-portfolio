PWA-Budget-Tracker

A simple budget tracker that works with or without an internet connection. This is a Progressive Web Application that implements service workers to enable loading and offline access. Users can enter in the amount to either deposit or deduct from their balance along with a description of the transaction.

Password Generator

A basic password generator using JavaScript. Allows for the user to create a random password based on criteria that the user chooses such as length and whether to include uppercase, lowercase, numbers, and special characters.

Random Employee Generator

Random employee generator that generates random employees using the randomuser.me API. The generator will present the user with employee photos, name, location, and email address. When the user refreshes the page a new set of employees will be generated
